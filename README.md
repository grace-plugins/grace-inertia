# Grace with Inertia

Grace Plugin for using Grace/Grails app with [Inertia.js](https://inertiajs.com).


## Development

### Build from source

```
git clone https://github.com/grace-plugins/grace-inertia.git
cd grace-inertia
./gradlew publishToMavenLocal
```

## Support Version

* Grace 2022.0.0+
* Grails 3.0+
